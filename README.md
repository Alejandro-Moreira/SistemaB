# Ecosistema AppleBox

Este repositorio contiene los componentes clave del ecosistema de seguridad de AppleBox:

## Estructura del Proyecto

- **[appleboxestudios-main](./appleboxestudios-main)**: Frontend desarrollado en React que implementa la interfaz de usuario y la integración con el SSO de Keycloak.
- **[micro-backend-b](./micro-backend-b)**: Microservicio en Node.js/Express encargado del descifrado de facturas y la integración con AWS KMS.

## Requisitos Previos

Consulte el archivo `README.md` de cada componente para obtener instrucciones detalladas de configuración y ejecución.
